# Flight Search Engine

Live Demo: [Live link](https://flight-search-assessment.vercel.app/)

## Features

- Real-time flight search using Amadeus API
- Interactive price graph
- Complex filtering (price, stops, airlines)
- Fully responsive design
- Smooth animations

## Tech Stack

- React 18 + TypeScript
- Vite
- Tailwind CSS
- Zustand (state management)
- Recharts (data visualization)
- Amadeus API
- AOS (animations)

## Setup

1. Clone the repository
2. Run `npm install`
3. Create `.env.local` with Amadeus API credentials
4. Run `npm run dev`
